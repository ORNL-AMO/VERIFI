A summary of the meter readings that are going to be uploaded is provided.

New readings will be added to the corresponding meter.

Invalid readings will be skipped. Check your import file for errors like missing read dates or energy consumption.

Existing readings have matching read dates with readings that are currently in the system for that meter and will overwrite the existing data when submitted.

Choose "Skip Existing" if you don't want to overwrite existing meter readings in the system.
