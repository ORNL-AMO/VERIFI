Manage Meters

All the utility meters for {{selectedFacility.name}} can be managed on this screen.

Click “Add New Meter” to add new utility meters.

Use “Import Data” to bulk import meter information in csv or excel format.

Once meters are added, they will be seperated by source. Use the tabs to navigate to those meters and add monhtly consumption data.

Manage Bills

All the {{selectedSource}} meters for {{selectedFacility.name}} can be viewed on this screen.

Use “Add new Bill” to add monthly {{selectedSource | lowercase}} consumption data data.

Use “Import Data” to bulk import {{selectedSource | lowercase}} consumption data in csv or excel format.

For electricity meters you can filter which columns you would like to track in the tables with the "Choose Columns" dropdown window.
