Data Entry Options

VERIFI has multiple options for adding utility data to corporate accounts:

  1. Direct VERIFI Data Entry: This option allows users to add data by filling out forms for each facility, meter, bill and predictor. This is a good option for users with less historic data, a small number of facilities, or users who are regularly updating.
  2. Excel Data Upload (Template): This option allows users to upload data for multiple facilities at once using an Excel template. This is a good option for users with many facilities or years of historic data.
  3. Excel Data Upload (Custom): Users who have data in a columnal format (columns for date, each meter, and each predictor) may be able to upload data without using the provided template. This option is best for users whose data can easily match the format and fields in VERIFI. (This option is good for legacy EnPI files.)
  4. Backup Data Upload (JSON): This option allows users to upload data from a JSON backup file. This is a good option for users who have exported data from VERIFI in the past and want to re-import it. This option can also be used to centrally manage facility performance data provided by facility energy managers.

Combinations of these options can be used. For example, a user could upload data for some facilities using the Excel template and then add additional data manually using the data forms within VERIFI.
