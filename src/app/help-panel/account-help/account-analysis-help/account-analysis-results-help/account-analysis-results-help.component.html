Analysis Results

This screen gives you the annual and monthly results of the account analysis. A rollup of each individual facility analysis.

Use the table/graph toggle switch to change between viewing the results as a detailed table or graphical representation of savings.

Use the "Choose Columns" dropdown to change the content included in the table.

The " Copy Table" button will copy the table data to your clipboard that can then be pasted into an excel file.

Click the " Create BP Report" to create a new Better Plants Report with this account analysis item.
