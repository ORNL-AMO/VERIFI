The utility meter data has been calendarized in this screen. Use the filter menu to view desired meters and over specific period. Data represented with N/A does not have a full month of data to calculate.

The utility meters can be grouped by utility type or user preference. These groups will represent your data visualization and regression analysis.

These group can be edited by dragging the meter from one group to another. The “Ungrouped” group will disappear once you have assigned all the meters to a designated group.

These are independent variables that will be used for {{ selectedFacilityName }}'s regression analysis. Predictors can be used to explain the change in your utility use.

Use “Add/Edit Predictors” to Add new Predictors for your analysis.

Use “Add Predictor Entry” to input your monthly predictor data.

Use “Import Data” to bulk import Predictor data in csv or excel format.
