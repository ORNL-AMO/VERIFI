Help

Help text is provided throughout the application with the use of a help panel. To open the help panel, click on the question mark () or arrow button () in the upper right hand corner.

Similarly, input fields have help text that can be seen by hovering over the corresponding question marks by the labels.

For additional help content please visit ORNL's Industrial Resources website https://industrialresources.ornl.gov/verifi. Here you will find a hub of content related to VERIFI including user guides, case studies, and information on ORNL's other tools in the VERIFI family.
