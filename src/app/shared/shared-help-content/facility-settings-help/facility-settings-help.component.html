Facility Settings

Use this screen to manage facility settings.

The units, sustainability goals, and financial reporting settings will be inherited from the account.

The Result Energy Unit determines the unit all your energy data is reported in. This can be changed here, at any time, to change the results in your Utility Data, Overview, and Data Visualization. You can set your result unit in analysis independently and this will not change any analyses or reports.

The Default Units only set the unit a new meter starts with. This can be set for each meter during meter setup. The Volume and Mass units are only used for setting the physical units of meters.

Your zip code is used to set the "Electricity Global Warming Potential", unless otherwise specified. This value is used to calculate the emissions for electricity meters in this facility. If you do not see this field, set your country to "United States of America (the)".

The answers to the Sustainability Questions will be used for reporting.

Use the financial reporting if your facility does not use a calendar year. The application will adjust to your fiscal year based on this setting.

Use the "Import Backup File" modal to overwrite the existing facility data with a backup file exported from VERIFI.
