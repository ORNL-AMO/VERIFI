Account Dashboard

On this screen you have a utility usage overview for the {{selectedAccount.name}} account.

The tables breakdown the data by Utility Usage, Utility Cost, and Emissions.

Data can be shown using Source or Site energy using the toggle buttons at the top.

The "Utility Usage", "Utility Cost", and "Emissions" will change which data is displayed in the graphs at the bottom of the screen.

The first table breaks down the consumption of each facility in the account.

Clicking on the facility name will take you to the summary for that facility.

The "Utility Use and Costs" table breaks down the consumption by utility. Showing the usage change from last month of data entered vs the data entered for one year prior.
