Select Facility Items

Use this screen to select the facility analysis items to include in this account analysis.

A table of the facility analysis items already conducted is provided. Use the radial buttons to select the item you want to include.

Click " Add Facility Analysis" button to add a new analysis for the selected facility.

Click "View Analysis" to go to the corresponding analysis item. You will be able to come back to this screen from that view.

A list of facilities in the account is provided. If a facility analysis item has been included there will be a check by that facility name.

Choose "Skip Facility" if for some reason you do not want to include the facility in this account analysis.
