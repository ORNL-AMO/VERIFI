The "Manage Predictors" tab can be used to add, edit, or delete predictors for this facility.

Use “Import Data” to jump to the import data tab for easy upload of bulk data in a csv or excel format.

Predictors are seperated by types. Either calculated or standard.

Use the pencil icon () to edit the predictor and the trashcan icon () to delete the predictor.

Use the temperature icon () in the calculated table to view more details about how that predictor is calculated.
