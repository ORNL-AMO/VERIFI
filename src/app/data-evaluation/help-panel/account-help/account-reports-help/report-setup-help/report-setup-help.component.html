Report Setup

Use this menu to setup your report. Provide a meaningful name, date range and choose the report type.

Depending on the report type you select the rest of the menu will change.

Automatically generate a Better Plants annual report for this account. Reports can be submitted to your Technical Account Manager for review and guidance.

For more information on Better Plants annual reporting see the Better Plants Energy Intensity Baselining and Tracking Guidance

For a better plants report, select the account analysis item you want to use. Click "View Analysis" to navigate directly to that analysis item.

The analysis items will show up based on the Report Year.

Some additional details, such as analysis notes, can be provided for the better plants report.

The graphs and tables included in the overview report are pulled from the account and facilities data overview tabs. The checkboxes can be used to select which ones you want to include in the report.

For the data overview report select the date range to filter the data to include.

Choose which facilities you want to include.

Choose which sections of the report to include.

Select an account analysis to get a detailed report of the performance of the account and its facilities using the models within the selected analysis.

Use the checkboxes under Section to Include to select how granular you want the report to go. View results by facilitiy, utility and/or group.

The content shown within the tables and graphs can also be modified by using the checkboxes under Report Content Options.

Select the years to include in the report.

Choose to include all utility data or to manually select which meter groups to include using the radio and checkbox options.

Customize the content included in the report with the checkboxes.

If using the report to fill out the Better Climate Challange form, "Emissions Reduction Iniciatives" can be added at the bottom of the setup and will be included in the report.

Select the years to include in the report.

A report with details on which emissions factors are used for each facility within the account will be generated.
