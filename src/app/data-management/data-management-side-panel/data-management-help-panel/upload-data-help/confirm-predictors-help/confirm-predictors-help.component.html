A summary of the predictors and predictor entries is provided.

Select if the predictor variables are production by using the toggle switch.

New predictor entries will be added to the corresponding facility.

Invalid readings will be skipped. Check your import file for errors like missing read dates or data.

Existing predictor entries have matching read dates with entries that are currently in the system for that facility and will overwrite the existing data when submitted.

Choose "Skip Existing" if you don't want to overwrite existing predictors entries in the system.
