Account Overview

On this screen you have {{overviewType}} overview for the data entered for {{selectedAccount.name}}.

Use the tabs to change the data overview type.

Use the dropdowns to filter the dates for which to include data. The last year of data entered is selected by default.

The first section of the overview provides a breakdown of the data by facility. The graph colors are representative of the facility color set in the settings.

A map of the facilities is provided based on their zip code. The size on the graph is based on the corresponding amount of consumption.

The {{tableType}} table shows the change in consumption against the previous year and against the monthly average over the selected date range.

A stacked bar chart of consumption by utility is provided for each facility for comparison.

Monthly consumption graphs are also provided for analysis.

Use the " Create Report" button generate a report of the current page.
