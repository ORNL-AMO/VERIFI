Manage Predictors

Manage and track independent variables that affect {{ selectedFacility.name }}'s utility consumption. Independent variables (or predictors) are used with regression analysis to model and normalize utility usage between years. Common predictors for manufacturing include production, cooling degree days, and heating degree days.

For more information on selecting independent variables see the Better Plants Energy Intensity Baselining and Tracking Guidance.

Use “Add/Edit Predictors” to add, rename, edit, or delete predictors from your analysis. Also use this to select which predictor is used as the production variable(s) for {{ selectedFacility.name }}'s utility analysis.

Use “Import Data” to jump to the import data tab for easy upload of bulk data in a csv or excel format.

Use “+ Add New Entry” to input new predictors for a new period.

Use the pencil icon () to edit the predictor entries in a row and the trashcan icon () to delete a row of predictor entries.

Use the “Copy Table” option to send predictor data to your clipboard to be pasted into external documents.

Use the toggle switch to indicate the predictor has been selected as a production item.
