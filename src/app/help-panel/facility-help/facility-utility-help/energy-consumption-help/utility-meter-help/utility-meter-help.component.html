Meter Reading Help

Use this screen to view and manage this meter's utility bills.

" Add New Bill" button will give you a form to add a new meter reading.

Clicking the "" button will allow you to edit existing meter readings.

Click the "" to delete the meter reading.

The "Choose Columns" dropdown can be used to select which columns you want to view in the table.

Use the checkboxes in the table to bulk delete readings. A "Bulk Delete" button will appear once one or more boxes are checked.

The " Import Data" button will take you to the Upload Data wizard where meters and meter data can be uploaded from excel files.

The " Copy Table" will copy the table data to your clipboard that can then be pasted into an excel file.

The "" button will export all of the meters and their data to VERIFI's excel template. This can be used as a backup or as a way to send data to others to be imported.
