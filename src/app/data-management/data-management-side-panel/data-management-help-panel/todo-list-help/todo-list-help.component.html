To Do List

A to do list has been provided for the selected account in the data management screen. This list includes items that will help you with the initial account setup as well as keeping things up to date.

Select the number of days that will be used to determine if data is outdated. The default is 60 days, but you can change it to any of the following options: 30, 90, 180, or 365 days.

Clicking on items in the list will take you to the appropriate screen to address the item.

If a weather predictor is out of date, you will see a button to update the weather predictors. This will automatically update the weather predictors for all facilities in the account to the latest data available.

Additionally, the sidepanel will also show a condensed version of the to do list that you can access at any time. Use the tab at the top of the side panel to switch to the To Do List tab.
