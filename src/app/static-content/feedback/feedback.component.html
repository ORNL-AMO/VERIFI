Feedback

Bug Reporting

To report a bug or otherwise provide feedback, please email Kristina Armstrong at armstrongko@ornl.gov. Please include your operating system (e.g., Windows 10, Mac OS 10.12), a screenshot of the issue and a download of your current data set so that we can understand the problem.

If you are using an international version of the application, please specify that when reporting bugs.

Thank you for your feedback!

Updating Issues

If you are being alerted of an update and are having issues downloading the latest update with the auto updater, please contact Kristing Armstrong with the link above. Additionally, you can go to our GitHub Page to directly download the latest update we have released. Installers can be found under the "Assets" section of the release.
