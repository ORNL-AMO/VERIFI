Data Entry Options

VERIFI has multiple options for adding utility data to corporate accounts. Users with only a couple facilities may want to skip this page and add data manually by filling out data forms for facilities, meters, bills and predictors. Otherwise, accounts with many facilities can use our excel template for uploading all the data tracking information at once.

  1. Manual Data Entry: This option allows users to add data by filling out forms for each facility, meter, bill and predictor. This is a good option for users with only a few facilities or for users who want to add data for a single facility at a time.
  2. Excel Data Upload (Template): This option allows users to upload data for multiple facilities at once using an excel template. This is a good option for users with many facilities or for users who want to add data for multiple facilities at once.
  3. Excel Data Upload (Custom): Depending on the structure of the data in the excel file, it may be possible to upload data without using the provided template. This option is best for users with a strong understanding of how the data is structured in VERIFI and how to map data from their excel file to the correct fields in VERIFI. (This option is good for ENPI files.)
  4. Backup Data Upload (JSON): This option allows users to upload data from a JSON backup file. This is a good option for users who have exported data from VERIFI in the past and want to re-import it, or want to selectively import facilities into the current account from a JSON backup file.

It should be noted a combination of these options can be used. For example, a user could upload data for some facilities using the excel template and then add additional facilities manually using the data forms.
