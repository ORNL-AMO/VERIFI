Edit Bill

Use this page to edit an existing meter reading.

Once reading data has been filled out click "Save" to submit changes and go back to the table.

New Bill

Use this page add a new meter reading.

Once reading data has been filled out click "Save + Quit" to submit and go back to the table. Otherwise click "Save + Add Another" to add another reading. The month will be incremented forward one month.

Hover over the "" icons for details about the corresponding input fields.
