Utility Meters

Use this screen to manage the meters in this facility.

" Add New Meter" button will give you a form to setup and add a new meter.

Clicking the name of the meter or the "" button will allow you to edit existing meters.

Clicking the "" button will create a copy of that meter.

Click the "" to delete the meter and it's corresponding readings.

The " Import Data" button will take you to the Upload Data wizard where meters and meter data can be uploaded from excel files.

Use the tabs to navigate to meter pages where you can manage individual meter bills.

The " Copy Table" button will copy the table data to your clipboard that can then be pasted into an excel file.

The "" button will export all of the meters and their data to VERIFI's excel template. This can be used as a backup or as a way to send data to others to be imported.
