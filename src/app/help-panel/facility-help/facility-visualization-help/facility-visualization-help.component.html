Data Visualization

Use the Data Visualization tab to analyze the data in this facility.

Select the date range of data you want to include in the visualizations and if you want to display individual meters or their groups.

The time series provides a line chart of the data entered over time. Choose which axis you want the variables to be plotted against.

The R2 variance chart provides a heatmap of the R2 values for all the pairings of the variables selected to be included.

The data correlation tab will provide a series of charts that show in depth details of selected x/y variable combinations. Details include a best fit equation, R2 and P-value.
