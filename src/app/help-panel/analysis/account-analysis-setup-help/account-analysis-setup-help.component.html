Analysis Setup Tabs

Use the “Analysis Setup” tab to:

  • Change the name of your analysis item
  • Select site or source energy for analysis
  • Change the common energy unit for analysis
  • Select the reporting year for analysis

Other options including the baseline year and reporting period range must be in the account “ Settings” tab.

Use the “Baseline Adjustments?” slider to enter account-level adjustments to account for issues such as changes in the number of facilities, unexpected events, or other operational changes relative to the baseline year.

For more information on adjustments including several common scenarios see the Better Plants Energy Intensity Baselining and Tracking Guidance.

Use the “Select Facility Items” tab to choose which facilities and analysis to include in the analysis report. A list of facility-level analysis for the selected report year in the “Analysis Setup” tab will appear for each facility entered.

An error icon () appears until at least one facility-level analysis is selected after which the “Analysis Results” tab can be selected.

Use the “Analysis Results” tab to view annual and monthly results for your account-level analysis.

Select between the “Table” and “Graph” views to see your results in more detail. Use the “Choose Columns” button to select which columns appear in the table view.
