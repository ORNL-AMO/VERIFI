Account Analysis Dashboard

Use the account analysis dashboard to manage your analysis items within this account.

Items created here can be used in the reports page to generate automatic reports for Better Plants Initiatives and other programs.

For more information on energy intensity baselining and tracking see the Better Plants Energy Intensity Baselining and Tracking Guidance.

The " Add New Analysis" button will create a new analysis item for this account.

Each card is representative of an analysis conducted on this account.

Use the "Show Details" toggle to show/hide details of each analysis within the cards.

The "" will denote that analysis item as the selected analysis for that year. The selected analysis will be used as default in other screens and the latest selected will be used to calculate savings towards your sustainablity goals.

The "" button will allow you to delete the corresponding item.

The "" button creates a copy of that analysis. This will include regression models and can be used to run analysis on different years using the same models.
