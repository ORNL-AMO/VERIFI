Use this page to upload data to this account.

A template has been provided to make uploading easier, use the download link to get the template.

Non-template excel file may be used (especially ENPI files) with some additional steps for uploading data.

Begin by choosing the files with data you want to upload to this account.

Multiple files can be uploaded at once. Each file will be processed one at a time in the order in which they are uploaded.

Once files have been chosen, use the button at the bottom of the page to begin working through the upload data process.
