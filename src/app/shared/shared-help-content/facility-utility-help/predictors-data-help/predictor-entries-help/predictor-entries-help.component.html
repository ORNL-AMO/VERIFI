The "Manage Predictors" tab can be used to add, edit, or delete predictors for this facility.

Use “Import Data” to jump to the import data tab for easy upload of bulk data in a csv or excel format.

Use the pencil icon () to edit the predictor entry and the trashcan icon () to delete the predictor entry.

Use the temperature icon () to view more details about how the values for that predictor is calculated.

Use the “Copy Table” option to send predictor data to your clipboard to be pasted into external documents.
