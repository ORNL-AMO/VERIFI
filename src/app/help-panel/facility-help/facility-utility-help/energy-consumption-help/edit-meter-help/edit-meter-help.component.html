Edit Meter

Use this screen to edit the properties of an existing meter.

Add Meter

Use this screen to fill out the properties of the new meter.

For energy meters “Site to Source” is also required. For grid purchased electricity, this value is usually 3, for fuels and on-site generated renewable electricity (wind or solar), the value is 1. For other cases, more information can be found in the Better Plants Energy Intensity Baselining and Tracking Guidance

Hover over the "" icons for details about the corresponding input fields.

Once all the details of the meter are correct, click the "Add Meter" button to submit the new meter.

Once the meter has been added it will show up as a new tab. Meter bills can then be added under that tab.

Once all the details of the meter are correct, click the "Save Changes" button to submit the changes to the meter.
