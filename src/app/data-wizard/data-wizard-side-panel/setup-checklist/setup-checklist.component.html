Getting Started Instructions

We have pulled the core data management pieces from the main portfolio area of VERIFI, into this data focused portion of the application. This was done in an effort to streamline the user experience for setting up and managing account data. Use the below steps and help content to assist in getting a new account setup. Once completed, the account portfolio can be used for analysis and data reporting.

1. Fill out corporate settings

Add general corporate settings like name and address of the account. Fill out any corporate goals for the account. These details are optional but will provide a better experience when using VERIFI for data reporting, emissions calculations and weather data.

2. Upload utility and predictor data

Data can be uploaded to VERIFI using excel spreadsheets. A template has been provided for easier upload. Download and fill out the template and then upload it to back into VERIFI to populate your account with facility, utility, and predictor data.

VERIFI does it's best to handle other formats of excel spreadsheets as well. Meter and predictor data can be uploaded if the worksheet is setup with date readings and usage in a column format.

3. Fill out facility details

Add general facility settings like name and address of the facility. Fill out any facility goals for the facility. These details are optional but will provide a better experience when using VERIFI for data reporting, emissions calculations and weather data. Data will be inherited from the account settings.

4. Add meters and meter data

Add meters and corresponding meter data for each facility. Once meters are added, group the meters together for analysis.

5. Add predictors and predictor data

Add predictors and corresponding predictor data for each facility. Predictors are used to conduct analysis and evaluate facility performance. Weather data predictors can also be automatically calculated by selecting a weather station closest to the facility and specifying which weather data to track.

6. Repeat steps 3 - 5 for all facilities

Repeat steps 3 through 5, adding facility details, meters, predictors and corresponding data. Once completed, step 7 will bring you to your portfolio to continue with analysis and reporting.

7. Go to portfolio for analysis and reporting

Once all data has been setup, click below to go to your corporate portfolio. In the portfolio you will get data visualization options, performance analysis and report generation.

Data management can still be done from the portfolio after data setup has been completed.
