Use this form to add or manage predictors you want to track for this facility.

Multiple types of predictors can be tracked. For direct value input, select "Standard". To have VERIFI calculate weather related values select "Weather".

The "Unit" will be used for display purposes in reports and analysis.

For weather predictors, select the type of weather data you want to track and the weather station to pull the data from. More details on weather data can be found in the "Weather Data" section of VERIFI.
