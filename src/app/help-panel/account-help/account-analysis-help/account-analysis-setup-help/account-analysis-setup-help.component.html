Analysis Setup

Use this screen to setup the details of this analysis item.

Give the analysis item a meaningfull name, it will be used in other areas of the application.

Select if you want the analysis conducted in source or site energy and what units to use for energy.

The report year is the year this analysis is to be conducted on.

Baseline year and reporting year are details set in the settings for the facility.

Use the " All Account Analysis" tab to return to the dashboard with all of this account analysis items.
