Account Overview

On this screen you have {{overviewType}} overview for the data entered for {{selectedAccount.name}}.

Use the tabs to change the data overview type.

Filter the data by date using the dropdowns. The last year of data entered is selected by default.

The first section of the overview provides a map of the facilities is provided based on their zip code. The size on the graph is based on their relative consumption, cost, or emissions.

Next is a breakdown of the data by facility. The graph colors are representative of the facility color set in the settings.

The {{tableType}} table shows the change in data for the last month compared to that of the previous year and the monthly average over the selected date range.

A stacked bar chart of consumption by utility is provided for each facility for comparison.

Monthly consumption graphs are also provided for analysis.

Use the " Create Report" button generate a report of the current page.
